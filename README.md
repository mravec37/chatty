# Chatty

Simple real-time chat application built with Spring Boot and WebSockets.

## Features

- Real-time messaging between users
- Unique username enforcement
- Online users list
- Private conversations
- Conversation history
- Archived conversations when user we chatted with disconnects
- Unread message indicator (blue dot)

## Tech Stack

- Backend: Spring Boot, WebSocket (STOMP)
- Frontend: HTML, CSS, JavaScript
- Communication: SockJS + STOMP
- Build tool: Maven

## Run

### Windows
start.bat

### Linux / macOS
./start.sh

Then open:
http://localhost:8080
